# Intro to Docker using Node.js

Project to practice Docker using Node.js 🐋📦

## How to run 🎲

Install Docker on your machine and run the following command in your project terminal:

```bash
docker-compose up
```

The API has 5 endpoints for a to-do app...
- List all to-do
- Get a to-do by id
- Update a to-do by id
- Create a to-do
- Delete a to-do
